# Bike Data Pipeline

## Overzicht
Een ETL-pipeline die fietsgegevens verzamelt, transformeert en opslaat voor verdere analyse.

## Functionaliteiten
- Data extractie uit de bron
- Data cleaning en transformatie
- Laden van de data in een database
- foutafhandeling

## Tech Stack
- Python
- PostgreSQL
- Telraam API
